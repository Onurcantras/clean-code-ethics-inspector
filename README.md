# Clean Code and Ethics Inspector (Gemini Gem)

Bu proje, yazılım geliştirme süreçlerinde kod kalitesini artırmak, Clean Code (Temiz Kod) prensiplerini uygulamak ve yazılım etiği standartlarını denetlemek amacıyla geliştirilmiş bir **Gemini Uzman Sistemi (AI Agent)** projesidir.

## 🚀 Projenin Amacı
Sistem Analizi ve Tasarımı kapsamında geliştirilen bu araç; yüklenen kod bloklarını analiz ederek okunabilirlik, performans, refactoring ihtiyaçları ve etik açıklar (güvenlik, veri gizliliği vb.) yönünden anlık geri bildirim sağlar.

## 🛠️ Nasıl Kullanılır?
1. Bu depoda yer alan `system_instructions.md` dosyasındaki talimatları kopyalayın.
2. Gemini platformunda (Gems / Kitaplık) yeni bir "Gem" oluşturun.
3. Kopyaladığınız talimatları yapay zekanın "Talimatlar" (Instructions) bölümüne yapıştırarak kaydedin.
4. Analiz etmek istediğiniz kod bloklarını doğrudan Gem'e göndererek denetimi başlatın.
